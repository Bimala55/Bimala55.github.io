About this project

In this project, we use plotly.js, a JavaScript data visualization library to build an interactive dashboard to explore the Belly Button Biodiversity dataset. A panel displays the sample's metadata, a bar chart of the top ten measurements is produced, a gauge chart shows the average belly button washes per week, and the entire dataset for this sample is at the bottom in a bubble chart.
